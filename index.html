🛡️ Remediation Hub

Policy Implementation

  • • Anti-Tamper: "Strict/On" (Cegah terminasi agent).
  • • Kernel AI: Aktifkan mode "Protect" pada driver engine.

Deep Visibility Query

IndicatorName = "VulnerableDriverLoaded" OR File.Name Contains "rwdrv"

Prevention Policy

"CS udah ada IOA untuk prevent BYOVD, pastikan enable settingan vulnerable driver protection."

  • • Policy: Aktifkan Vulnerable Driver Protection.
  • • Anti-Tamper: Set ke level "Aggressive".

KQL Hunt

DeviceImageLoadEvents | where FileName =~ "rwdrv.sys"
